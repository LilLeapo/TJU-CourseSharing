杨雅君部分把PPT全背熟，全是证明题，会考你怎么证明图灵机的一些性质，难度约是离散数学的两倍。

刘志磊部分就是计算题，没有证明题，大可不必看他的证明。
就是给你数字，计算求得数，基本上就是线代知识，你要复习一遍线代才行。
算的很恶心，方法很简单，但是不一定能写对。
大概率还是原题，数可能会变，但是题不会变。
